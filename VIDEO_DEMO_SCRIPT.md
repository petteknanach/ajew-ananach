## 🎥 Production Notes

### Technical Requirements:
- **Resolution:** 1920x1080 (1080p)
- **Frame rate:** 60fps for smooth animations
- **Aspect ratio:** 16:9 (landscape) or 9:16 (vertical for social media)
- **File format:** MP4 with H.264 codec

### Device Simulation:
- Use iPhone 15 Pro simulator (393x852 points)
- Show device frame for context
- Smooth screen transitions
- Highlight touch interactions with circles

### Audio:
- **Voiceover:** Clear, calm, spiritual tone
- **Background music:** Soft, inspirational instrumental
- **Sound effects:** Subtle tap sounds, page turns
- **Volume mix:** Voiceover 70%, music 30%

### Animation Style:
- **Entrances:** Smooth fade and slide
- **Transitions:** Cross-dissolve between scenes
- **Focus effects:** Subtle zoom on important elements
- **Text animations:** Clean fade-in

## 🛠️ Recording Setup

### Software Options:
1. **ScreenFlow** (Mac) - Professional screen recording
2. **OBS Studio** (Free) - Open source recording
3. **QuickTime** (Mac) - Simple screen recording
4. **Android Studio Emulator** - For Android preview

### Recording Steps:
1. Launch iOS Simulator (iPhone 15 Pro)
2. Load Ajew Ananach app via Expo
3. Set recording area to simulator window
4. Record at 60fps, 1080p resolution
5. Capture clean audio separately
6. Edit in Final Cut Pro / Premiere Pro

### Editing Notes:
- Add intro/outro animations
- Include app store badges
- Add subtle text overlays
- Sync voiceover with visuals
- Add background music
- Export multiple versions
